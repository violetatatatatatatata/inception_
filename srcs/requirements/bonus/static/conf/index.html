🚀 System Overview

This project consists of a complete system infrastructure based on Docker. Every service runs in its own containerized environment, following 42 Network's strict security and performance standards.

NGINX (TLS 1.3)
WORDPRESS
MARIADB
REDIS CACHE
FTP SERVER
ADMINER

🛠️ Implemented Bonuses

  • Redis: High-performance object cache to reduce database load.
  • Adminer: Web-based database management tool.
  • FTP (vsftpd): Secure file transfer protocol pointing to WordPress volume.
  • Static Site: This very dashboard, served via a dedicated Nginx instance.
